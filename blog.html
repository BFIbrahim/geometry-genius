Document Object Model (DOM). DOM is a programming interface and with this we can change the content, structure of a html, style. The purpose of DOM is a developer can easily manipulate the content stracture of a web page.We can easily add,remove,update. With a click button we can do many things as like change the style, set a data from one field to other field. DOM is as like a tree. each object of tree is element or node. That means anything found in HTML and XML we can be accessed.

There are five ways we select the element using DOM.
1.getElementsByTagName()
2.getElementsByClassName()
3.getElementById()
4.querySelector()
5.querySelectorAll()

Event bubble is the concept of Document Object Model (DOM).When a element receive an event then the event bubble up to parents. and its works in the DOM tree until its gets to root element.

A callback function is a function which is executed after another is finished to execution. That means a function can pass arguments to another function so that it can be executed to another function. Callbacks are used to handle the results of asynchronous operations, which means that the operation does not block the execution of the rest of the program
